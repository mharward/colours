# Colour Converter

A lightweight web tool for converting between colour formats.

## Features

- Paste any colour code (HEX, RGB, RGBA, HSL, HSLA)
- See instant conversions to all formats
- Visual colour preview
- One-click copy to clipboard

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed automatically to Netlify from the main branch.
